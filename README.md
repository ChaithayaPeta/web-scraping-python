# Data Analysis by Web Scraping Using Python

This project scrapes book data from an online shopping website and analyzes it using Python.

## Tools Used
- Requests
- BeautifulSoup
- Pandas
- Matplotlib

## Features
- Multi-page scraping
- Price comparison (cheapest & most expensive books)
- Data visualization (ratings & price analysis)
- Category-wise scraping
